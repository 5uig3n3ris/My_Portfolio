# My_Portfolio
# 🌐 Personal Portfolio Website

A modern, responsive, single-page portfolio website showcasing my projects, technical skills, career aspirations, and contact information. The website provides a clean and intuitive user experience while demonstrating front-end web development best practices using HTML, CSS, and JavaScript.

---

## 📖 Overview

This portfolio serves as a personal website designed to:

- Introduce who I am
- Showcase selected projects
- Highlight my technical skills
- Share my educational background and career aspirations
- Provide a simple way for visitors to contact me

Although the website is built as a single-page application, JavaScript is used to create a smooth multi-page browsing experience by displaying one section at a time.

---

## ✨ Features

### 🏠 Hero Section

- Professional introduction
- Brief personal summary
- Call-to-action buttons
- Smooth navigation to projects and contact sections

### 💼 Featured Projects

Displays selected portfolio projects including:

- Project title
- Description
- Technology stack
- Live Demo button
- GitHub Repository button

### 👤 About Me

Contains information about:

- Educational Background
- Career Aspirations
- Technical Skills
- Hobbies & Interests

### 📬 Contact Form

The contact form includes:

- Name
- Email Address
- Phone Number
- Message

JavaScript validation ensures:

- All fields are completed
- Email address follows a valid format
- Phone number contains only digits

### 🎨 Responsive Design

- Mobile-friendly layout
- CSS Grid and Flexbox
- Responsive navigation
- Adaptive project cards
